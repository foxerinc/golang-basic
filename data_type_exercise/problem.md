# Data Type Exercise — Problem

## Objectives

- Practice choosing appropriate data types for different kinds of data.
- Explain the assumptions and reasoning behind each choice.

## Description

For each of the questions below, pick a suitable data type and explain **why** you chose it and what assumptions you are making.

You can answer in general terms (integer, float, string, boolean, etc.) or using concrete language types (e.g. `int`, `float64`, `string`, `bool`) depending on the context you are working in.

## Questions

1. **Person's age**  
   What would be the appropriate data type for storing a person's age?

2. **Number of pages in a book**  
   What data type would be appropriate for storing the number of pages in a book?

3. **Price of a meal**  
   Which data type would you use to store the price of a meal at a restaurant?

4. **Total population of a city**  
   What data type would you use to represent the total population of a city?

5. **Grade Point Average (GPA)**  
   Determine the suitable data type for representing a grade point average (GPA).

6. **Email address**  
   Identify the appropriate data type for storing an email address.

7. **Accepted terms and conditions flag**  
   What data type would be the most appropriate for storing a flag that indicates whether or not a user has accepted the terms and conditions?

8. **Distance from the Earth to the Moon**  
   What type of data would you use to store the distance from the Earth to the Moon?

9. **ISBN number of a book**  
   Determine the suitable data type for representing the ISBN number of a book.

10. **Hexadecimal color value**  
    Identify the appropriate data type for storing the hexadecimal value of a color.

11. **Movie rating (1–5 stars)**  
    Determine the suitable data type for representing the rating of a movie (1–5 stars).

12. **File size of an image in bytes**  
    Identify the appropriate data type for storing the file size of an image in bytes.

## Instruction

For each item:

- Choose a data type.
- Write 1–3 short sentences explaining:
  - Why that type fits.
  - Any assumptions you are making (for example: “we only store whole years”, “we need decimal precision for money”, etc.).
